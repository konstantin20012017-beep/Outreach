VPS парсера (`85.137.165.185`, SmartApe CZ) — root-пароль агенту не передаётся. После ротации 2026-05-15 пользователь не делится новым паролем. Запись `VPS_PASSWORD` в `.env` проекта Парсер — устаревшая, не использовать для SSH.

**Why:** Принцип безопасности из CLAUDE.md проекта — «секреты на сервер заливает пользователь сам, агенту не показываем». Распространяется и на root-доступ к VPS.

**How to apply:** Когда нужно посмотреть `/var/log/parser.log`, рестартнуть `systemctl restart parser`, обновить файлы в `/opt/parser/` — не пытаться SSH'нуться сам, а давать пользователю готовую команду для копипасты и просить вывод. Если правка большая (новый файл, патч кода) — готовить локально + инструкцию для `scp`/`rsync`, которую пользователь запустит сам.
