`справочники/фундамент-текстов.html` — наглядная дайджест-карта всего, на что опирается пишущий агент (красные линии, оба skill'а, формат, глубокая персонализация). Это **зеркало**, а не источник.

**Источник правды (правило 2 архитектуры):**
- `.claude/skills/аутрич-стратегия/SKILL.md` — «как писать» (сегментация, триггеры, формулы, мимикрия, бенчмарки).
- `.claude/skills/аутрич-цепочка/SKILL.md` — процесс письма (роли, CTA, разводка, темы, структуры тела).
- `.claude/agents/аутрич-копирайтер/аутрич-копирайтер.md` — сам агент.
- `справочники/спам-стоп-слова.md` — данные (205 спам-слов), заведены 2026-06-04.

**How to apply:** правишь правило в skill — обнови `фундамент-текстов.html` тем же содержанием в той же сессии. Иначе дайджест отстанет от skill'ов и начнёт врать (нарушение правила «единый источник правды»). При сверке «есть/нет» читать первоисточник (skill), не HTML — HTML может отставать.

**Why:** на сессии 2026-06-04 завёл паттерн двойной записи (skill + HTML). Без явной фиксации следующая сессия рискует поправить только skill и забыть зеркало.
