# iso-smk (СМК Стандарт) — кейс-референс по нише сертификации/СРО/НОК

Референс при проектировании цепочек по сертификации/СРО/НОК.

- **Файл:** `брифы/iso-smk_кейс.md` + `.html` (анализ отчёта `Отчет СМК.xlsx`, 5 кампаний)
- **Лидеры:** ФЗ-309 «На тех, кто уже в СРО» — 29 лидов абсолют (триггер 1 марта); СРО-71 проектирование — reply 9.2% (цифры УТП в письме №1: 5000₽/6 мес без взносов/своё СРО); сертификат соответствия по узкой базе 54 контакта — 33% interest (триггер дата окончания)
- **Главные инсайты:** (1) реальный регуляторный триггер с датой = бомба; (2) узкая база с дата-окончания в 30× эффективнее широкой; (3) цифры в УТП в письме №1 = в 3.4× выше reply; (4) бесплатная проверка как pre-sale = низкий барьер; (5) НОК сама по себе слабая, в связке с ФЗ-309 — топ; (6) квалифицирующий вопрос в конце письма №1 — стандарт ниши
- **Применение:** содержание (триггеры, цифры, формулировки) — забираем; формат (темы, `{{доброгоВремениСуток}}`, 2 письма) — НЕ забираем, у нас стандарт CLAUDE.md
