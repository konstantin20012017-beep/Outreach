В отклонённых Haiku сообщениях (фильтр парсера «Авто из Китая») регулярно встречаются запросы на ремонт, диагностику, запчасти, сервис — это потенциальные лиды для авто-сервиса/магазина запчастей, не для текущего клиента (импорт из Китая).

**Why:** пользователь увидел это сам при просмотре логов 2026-05-19. Сейчас клиента под эту нишу в работе нет, но рынок очевиден. Это бесплатный побочный поток — материал уже фильтруется парсером, нужно только отдельный канал выдачи и адаптированный промпт.

**How to apply:**
- Когда появится клиент в нише «ремонт авто / запчасти / автосервис» — поднять идею второго ниша-канала в текущем парсере. Архитектура уже это поддерживает (см. CLAUDE.md, раздел про конфиг ниш).
- Не предлагать самому делать это сейчас — клиента нет, [[feedback-dont-re-ask]] и [[feedback-no-premature-features]] (без преждевременных фич).
- При проспектинге услуг аутрича — авто-сервисы СПб/МСК это релевантный сегмент, у нас уже есть готовый источник лидов как демо.
