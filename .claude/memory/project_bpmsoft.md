- **Спикеры:** Оксана Репина (Группа СЕБ-ВОСТОК / Tefal-Moulinex, эффективность +30%), Максим Пономарев (ВОЛМА, БП x5 — формулировка только с лендинга, в кейсах есть лиды -40%, закупки -20%, КП с дней до 2 часов), Сергей Петров (BPMSoft)
- **ЦА:** одна, без отраслевого деления. Средний и крупный бизнес от 1-2 млрд/год, РФ+РБ. Должности: владельцы/CEO/COO, коммерческие/маркетинговые директора, ИТ-директора. Финальный список приоритетных отраслей Андрей пришлет
- **Структура работы:** A/B-тест из 3 цепочек на одной нише с разными заходами:
  - Цепочка №1 — по референсу LeadBridge/Coldy (49 регистраций за 2 недели). v3 — `готовые цепочки/bpmsoft_цепочка1_рефLeadBridge_v3.html`. 2 письма, БЕЗ ссылки в письме (отправляется отдельно тем, кто ответил «+»), с темой в теле, плейсхолдеры `{{firstName}}`. №1 общая программа, №2 «поднимаю переписку» + конкретика по спикерам с цифрами
  - Цепочка №2 — заход через бренды и регалии. v3 — `готовые цепочки/bpmsoft_цепочка2_бренды_v3.html`. 3 письма, ссылка в письмах. №1 регалии (1 место TAdviser/CNews/Сколково) + бренды (ГАЗ, ВОЛМА, ЦУМ, VK, Билайн, Газпромнефть, ТеДо, Б1). №2 спикеры с цифрами + «у других клиентов BPMSoft» (Билайн +24% чек, VK -60% отчеты, Аскона x1.5 онлайн, Туту 3 часа). №3 короткий follow-up + PS «перешлите коллеге»
  - Цепочка №3 — заход через миграцию с зарубежного ПО + ROI. v3 — `готовые цепочки/bpmsoft_цепочка3_миграция_ROI_v3.html`. 3 письма. №1 — список миграций (ЦУМ за 2 недели + ТеДо 750+ мест + Б1 1052 места + ППР/Биокодекс/СберФакторинг/GEFCO) + ФСТЭК. №2 — каждая ROI-цифра привязана к компании (ППР цикл -2-3 раза, Аскона x1.5, Билайн +24%, VK -60%, Туту 3 часа) + общие (маркетинг -50%, IT -40%). №3 короткий follow-up
- **Финалки v3:** все 3 цепочки за 5 рабочих дней дали **0 регистраций**. Гипотеза: все три открывались с «приглашаю на вебинар» (Solution Aware), а ЦА на Problem Aware. См. [[feedback_match_hunt_awareness_level]]
- **Цепочка №4 — pain-first, v1:** `готовые цепочки/bpmsoft_цепочка4_pain_first_v1.html`. 3 письма с разными углами из 5 пунктов программы лендинга: №1 — разрозненные системы + руководитель не видит бизнеса; №2 — скорость работы с лидами + неполный возврат маркетингового бюджета (кейсы ВОЛМА +40% / Аскона x1,5 / Билайн +24%); №3 — сроки согласований КП и закупок (ВОЛМА закупки -20% / КП с дней до 2 часов, СЕБ +30%). Подача широкая (Hunt 1-2), без выдуманных сцен. PRE-FLIGHT с разделением [ИСТ]/[КАТ]/[СЦЕНА] (см. [[feedback_pain_category_vs_scene]])
- **Подача:** от лица менеджера маркетингового партнера BPMSoft. Не «интегратор», не сам BPMSoft. «являясь маркетинговым партнером, приглашаем на вебинар BPMSoft»
- **«Без зоопарка систем» — название вебинара клиента**, не выдумка. Использовать в кавычках
- **«БП в 5 раз» (ВОЛМА)** — формулировка ТОЛЬКО с лендинга, в PDF «Истории Успеха» этой цифры нет. Подавать как речь спикера: «расскажет, как ускорили БП в 5 раз». Не утверждать как факт
