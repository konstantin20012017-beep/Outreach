# RefExpress (refexpress.ru) — БУ реф-контейнеры

В работе.

- **Бриф:** `брифы/refexpress_бриф.md` + `.html` (под рассылку). Сырьё: `брифы/refexpress_исходники.md` (созвон 2026-05-18 + парсинг сайта + материалы клиента по сегментации)
- **Юрлицо:** ООО «Р.Э.С.», СПб. На рынке с 2012 (14 лет). Регалии с сайта: 800+ контейнеров/год, импорт 300+, парк 100+ дженсетов, клиенты из 67 регионов РФ + Беларусь + Казахстан
- **Что продаём:** БУ-реф-контейнеры. Кросс-сейл в №2/№3 — запчасти + дизель-генераторы (прямая просьба клиента, формулировка в чате «запасные части и моторы»). Аренда — лёгкий упомин. Новые / тендеры в аутрич не берём
- **«Моторы» — открытое поле:** на созвоне разделялись запчасти и генераторы как два продукта, в чате клиент сказал «моторы» — может быть синоним генераторов или отдельная категория (рефрижераторные агрегаты/компрессоры)
- **Сегментация:** 1 — конечники (фокус, пишем цепочки), 2 — допы запчасти/генераторы (cross-sell в №2/№3, отдельных цепочек нет), 3 — партнёры (на потом). На Сегмент 1 несколько цепочек для A/B-теста
- **Сегмент 1 — конечники, ~15к компаний (проспектинг Иванова, ЮЛ):** рыболовство 3225 / мясо-птица 4000 / молочка 2400 / заморозка-полуфабрикаты 1800 / овощи-фрукты-ягоды 4000. Доп. отрасли (меньший приоритет): нефтегаз, логистика, химия, крупные торговые сети с хабами. НЕ берём: общепит, ритейл продуктов, кондитерка, напитки, холодильник для воды
- **Критерий ЦА:** склад готовой продукции с охлаждением. География — вся РФ, 12 городов-миллионников (клиент подтверждает 12; на сайте 6 собственных терминалов — СПб-Шушары, Новосибирск, Красноярск, Иркутск, Хабаровск, Владивосток — остальные точки выдачи/партнёры, нужен полный список)
- **ЛПР:** закупщики, инженеры, собственники, директора
- **УТП:** (1) логистика — 12 городов, доставка с ближайшего, бесплатно в Урал/Сибирь/Якутию/ДВ, своя ТК (конкуренты — самовывоз); (2) свой сервис, гарантийный/постгарантийный ремонт, PTI, восстановление поршневых компрессоров, круглосуточный сервис в Шушарах; (3) гарантии 90 дней БУ / 3 года новые / 30 дней запчасти; (4) запчасти Carrier/ThermoKing/StarCool/Daikin + аналоги — свой склад; (5) видео-обзор по запросу (страх БУ из другого города — не битые, не «после рыбы»); (6) полный пакет документов (Carrier как производитель — в письме нейтрально из-за санкций, но как бренд запчастей упоминать можно)
- **Триггер:** летний сезон — пищевики заготавливают, заранее озаботиться. Боли — 4 статьи на сайте `/articles/`
- **Что спрашивают входящие:** наличие / количество / состояние. **CTA:** «пришлите номер — проверим наличие». Вариации: количество+город / видео-обзор / ИНН для проверки ближайшего терминала
- **Подпись писем:** Алексей Пасюта (телефон/email/TG/должность — открытое поле)
- **Контекст клиента (не для писем):** amoCRM, каналы Я.Директ+SEO+автопрозвон по ОКВЭД+Авито, база реанимации 3000, домены вымышленные, цикл сделки 1-3 мес, квал. лид — нужен в 6 мес + бюджет от 600к ₽
- **Открытые поля:** Алексей Пасюта (полные контакты); что под «моторами»; кейсы и отзывы (Алексей обещал поспрашивать); файл на Я.Диске https://disk.yandex.ru/i/bTIwopaULJlo6w (упомянут в графах отраслей и географии — попросить дублировать); полный список 12 городов
