# NUMIS — управление продажами (SaaS)

**Что:** аналитическая платформа управления продажами. Концепция «Статистик и Состояний»: определяет состояние продаж (рост/застой/кризис) + выдаёт рекомендации. Обещание: «руководитель за 5 минут понимает, что происходит и что делать — один фокус, не 20 метрик». Слоган: «аналитик + консультант + РОП без найма».

**Сайт:** numi.plus (главный) + promo.numi.plus/razbor-prodazh (лид-магнит «разбор продаж за 30 мин»). Юрлицо ООО «Платформа», Нижний Новгород. Лицо: Евгений Дубровин, собственник (проводит разборы лично).

**Рынок:** B2B РФ, десятки тысяч компаний с отделом продаж от 3 менеджеров. Средний чек 15 тыс. ₽ (тарифы сайта 5 990 / 17 990 / 22 990 ₽/мес — расхождение уточнить).

**Сегментация (2 оси):** по роли ЛПР (собственник / РОП / комдир) ИЛИ по размеру отдела (3-5 / 6-10 / 10+) — обе готовы на лендинге. Какую ось в цепочки — решает клиент.

**Боли (сорсированы с лендинга):** план не выполняется, непонятно что мешает; отчётов много, ясности нет; команда работает неровно, просадка видна поздно. Мета-боль: «CRM хранит данные, но не подсказывает решение» → NUMIS = слой решений поверх CRM.

**CTA-якорь:** бесплатный разбор продаж за 30 мин (готовый лид-магнит, проводит собственник).

**Бриф:** `брифы/numis_бриф.md` + `.html`, сырьё `брифы/numis_исходники.md` (текст 2 страниц сайта + анкета, 2026-06-24).

## Стратегический созвон 2026-06-24 (вечер) — тактики
Транскрибация в `numis_исходники.md`. Решения:
- **3 тактики аутрича = оси цепочек:** (1) ICP+кейс по нише (массовее); (2) триггер-вакансии (кто набирает менеджеров продаж, оборот от 200 млн, теплее); (3) ЛПР по должности (резюме-базы, РОП основной). Под нишу — своя цепочка, малые списки 500-1500 контактов.
- **Общая почта vs ЛПР:** в РФ общие почты в крупном бизнесе работают лучше. Заход на общую почту НЕ через прямую боль (секретарь = реклама), а «у нас релевантный кейс, передайте коллеге». ЛПР-почта — прямая боль. → A/B уместен.
- **CTA мягкий (клиент согласовал):** письмо НЕ закрывает на демо. №1 = лид-магнит/разбор (низкий барьер) → телефон → квал → демо. Якорь — бесплатный разбор 20-30 мин.
- **Формат кейса в письме («матрица кейса»):** 3 абзаца — «видим вы в сфере X / есть кейсы по нише / клиент пришёл с проблемой → результат [цифра]». Сюда ложатся +10%/8%→11%, сфера без имени.
- **Отстройка №1 — от CRM** (не Excel): «у вас есть CRM → мы дополняем, показываем что делать». Excel второй.
- **Инфра:** ~1500 писем/раб.день, цепочка 3 письма → 30k писем/мес, база 10k. Отклики 50/25/25 по письмам → письмо №1 решает.
- **Процесс:** утвердить сегменты → копирайтер → согласование → через 2 недели запуск.
- Ждём от стратега пример письма с кейсом; от клиента — матрицу кейсов по нишам.
