# DITEC (pro.ditec.ru) — машиностроение (3 продукта)

В работе. 3 продукта = 3 сегмента/цепычки, все v1 готовы.

- **Бриф:** `брифы/ditec_бриф.md` + `брифы/ditec_бриф.html`
- **3 продукта = 3 сегмента/цепочки:** (1) линии ламинации/окутывания PUR, (2) станки нарезки рулонов, (3) орбитальные упаковщики стрейч
- **Сегмент 1 — линии ламинации (PUR Line):** цепочка v1 готова (`готовые цепочки/ditec_ламинация_v1.html`)
- **Сегмент 2 — станки нарезки рулонов (DIV):** цепочка v1 готова (`готовые цепочки/ditec_нарезка_v1.html`). Цена от 1 158 000 ₽
- **Сегмент 3 — орбитальные упаковщики (Techpack):** цепочка v1 готова (`готовые цепочки/ditec_упаковщики_v1.html`). Цена от 1 400 000 ₽, экономия пленки 50%
- Все 3 цепочки: без конкретных кейсов от клиента (только регалии завода — 400+ фабрик, 30+ млрд оборот, 15+ лет). Плейсхолдер подниш в письме №1 — варианты в [скобках через слеш], оператор выбирает один
- **Завод:** ООО «Дитек машиностроение», Тольятти. 15+ лет, 400+ клиентов, 30+ млрд оборот клиентов
- **Контакт:** Евгений (созвон), РОП Ростислав Тункин
- **CTA:** основной — КП + ТТХ; в 1-м письме мягче (каталог / видео-обзор / референсы / экскурсия). Цикл сделки длинный, клиент может планировать через полгода
- **Запросы клиенту:** 1-2 ключевые боли по каждому продукту, кейсы с цифрами, список именитых клиентов, готовое КП
