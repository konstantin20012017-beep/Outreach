Whisper (включая API и публичные боты) даёт **плавающие word timestamps** и часто разметит слово с длительностью 0 — особенно короткие частицы вроде «чтобы», «как». Это приводит к неточным вырезам в монтаже: либо клипится соседнее слово, либо вырез наезжает на нужное.

**Why:** На проекте «Монтаж ролика» мы потратили несколько итераций, пытаясь исправить вырезы поверх whisper-транскрибации, прежде чем перешли на Deepgram Nova-3 — и сразу всё встало точно. Пользователь спросил «почему сразу не сказал».

**How to apply:** Когда задача требует word-level таймкодов (видеомонтаж по словам, синхронизация субтитров, форсированные вырезы) — сразу рекомендовать **Deepgram Nova-3** ($0.43/час, у пользователя есть $200 кредита) или AssemblyAI ($0.37/час). Не предлагать Whisper API как «то же самое» — оно не то же. Forced alignment локально (WhisperX) у пользователя вешает мак, не пробовать.
