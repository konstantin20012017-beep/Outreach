# Грандпроект (ООО «Гранд Проект», stroy1c.ru)

Внедрение/доработка 1С для строительных компаний («Цифровое строительство»), Москва. Стек: 1С:Подрядчик строительства (ядро), ERP, УАТ, БСП, Мобильный клиент, +ИИ-модуль «Антидубли» (ловит дубли материалов). Регалии: 30 проектов, 17+ лет, ТОП 40 рейтинга 1С по Москве, с 2006.

**Бриф v1 2026-06-29:** `брифы/грандпроект_бриф.md` + `.html` + `грандпроект_исходники.md` (12 кейсов с /proekty/ вытащены дословно через firecrawl, охват полный).

ЦА: стройкомпании — генподряд полного цикла, монолитчики, механизация/спецтехника, закупочные площадки. Клиенты кейсов: СУ-90, СК «СУ» (генподряд Москвы для МР-Групп/October/AFI/ИНГРАД/Level/Абсолют), Закупка.про, Служба механизации.

**Боли по функциям:** снабжение (избыточные закупки/потери), финансы (переплаты/просрочки), сметы (долго+ошибки), автотранспорт (ручные путевые/топливо), стройпроект (старт работ без РД→срыв сроков), кадры (штрафы за патенты/мошенничество с табелем), ОТ и ТБ, производство.

**Кейсы стройки в основном БЕЗ цифр** («в несколько раз», «обнаружены переплаты»), твердая только: автотранспорт (Служба механизации) — штат на путевых **с 5 до 1 чел**, ~120 листов/день.
