# Реестр результатов кампаний

Единый источник правды по тому, **что сработало**. Из него цепочка получает статус победителя. Заполняется по мере возврата данных от клиентов.

## Поля
`Проект | Сегмент/цепочка | Файл | Ниша | Тип оффера | Awareness | Период | Объём | Open % | Reply % | Лиды | Вердикт | Почему | Источник`

> «Источник» обязателен: `отчёт клиента` / `xlsx` / `со слов` / `из памяти`. Бэкфилл по памяти помечаем честно — это не то же, что выверенный отчёт.

### Сертификация / декларирование / СРО (iso-smk, параллельный клиент)

| Кампания | Объём | Reply % | Lead % | Вердикт | Почему |

| ФЗ-309 «уже в СРО» | 3 355 | 4.5% | 0.86% | ⭐ лидер по лидам (29) | регуляторный триггер с датой + 5 рисков в цифрах |
| Сертификат (узкая база) | 54 | 100% | 33.3% | ⭐ топ по конверсии | фильтр «истекает документ в этом месяце» |
| СРО-71 проектирование | 1 028 | 9.2% | 1.95% | ⭐ топ по reply | цифры УТП в первом письме (×3.4 к СРО-41 без цифр) |
