# Лог рынка аутрича

Датированный append-only лог свежих тактик холодного аутрича: дэливерабилити, опенрейт, инструменты, изменения почтовых провайдеров, находки маркетологов (РФ + зарубеж).

Правило: **не переписываем старое, только дописываем сверху с датой.** Раз в квартал устойчивое переносим в skill `аутрич-стратегия` (помечаем `→ КАНДИДАТ В STRATEGY`).

## Формат записи
```
## YYYY-MM-DD — <тема>
Суть: <что за тактика/изменение>
Источник: <URL> (дата публикации, строка доверия: кто/аудитория)
Применимо: <ниша / тип оффера / awareness>
Статус: гипотеза | подтверждено метрикой в реестре | → КАНДИДАТ В STRATEGY
```
