# Библиотека кейсов

Переиспользуемые **citable кейсы** — истории «компания X внедрила Y, получила Z», которые копирайтер вставляет в письмо как доказательство.

Не путать с победителями: победители — это «какая цепочка сработала», кейсы — это «какой результат у клиента можно процитировать».

## Формат файла кейса
Один файл на кейс или на нишу. Внутри по каждому кейсу:
- **Отрасль**
- **Что было** (проблема)
- **Что внедрили**
- **Результат** — цифра
- **Источник + дата** (обязательно, красная линия 2)
- **Близость к нишам** — где уместно цитировать

## Дисциплина
- Только реальные кейсы с источником. Нет источника — кейса нет.
- Кейс из смежной отрасли цитируется как смежный, не как «у вас в отрасли».
- Свежесть проверяется по дате источника.
