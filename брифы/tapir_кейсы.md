# Кейсы Тапир

## Kitogo (kitogo.ru)
- **Сфера:** Одежда
- **Услуга:** SEO, Разработка, UX/UI
- **Запрос:** Новый интернет-магазин без трафика, заказов и SEO фундамента. Получать стабильные заказы из органического поиска, запустить SEO с нуля без вложений в рекламу
- **Результат за 3 месяца:**
  - С 0 до 5 000+ органических пользователей в месяц
  - +268% среднемесячный рост трафика
  - Заказы выросли в 5 раз
  - Индексация увеличилась в 3,7 раза
  - Средняя позиция: 6,3
  - CTR: рост в 2,1 раза
  - Глубина просмотра: +70%
  - Повторные визиты: +34%
  - Время на сайте: 3:40
- **Эффект:** Сайт начал приносить прибыль без рекламы, сформировалась лояльная аудитория, SEO стало стабильным каналом продаж

## Waistline (waistline.shop)
- **Сфера:** Одежда (женский бренд одежды и белья)
- **Услуга:** Разработка, UX/UI
- **Запрос:** Разработать интернет-магазин, который по удобству не уступал бы маркетплейсам. Дать возможность работать с клиентской базой, управлять LTV, создавать уникальный функционал
- **Что сделали:**
  - Удобный подбор размера (вводишь измерения – сайт подсказывает размер)
  - Каталог с фильтрацией по степени утяжки, эффекту, коллекции
  - Карточка товара с фото размеров на разных моделях, видеообзорами
  - Корзина с расчётом до бесплатной доставки, подарки, промокоды, баллы
  - Агрегатор доставки: курьер или ПВЗ с примеркой
  - Бандлы – экономия на покупке наборов
  - Личный кабинет с программой лояльности
- **Эффект для бизнеса:**
  - Возможность работать с базой и управлять LTV
  - Создавать отличный от МП функционал и собственные сервисы
  - Разнообразнее каналы трафика
  - Усиливать собственный бренд за счёт мультиканальности
  - Более широкие аналитические возможности для увеличения ROI
- **Известный бренд среди селлеров одежды** – кейс перехода с Marketplace на собственный e-com
