- **Сайта и домена** — не дан (точное название бренда тоже под вопросом).
- **Документа «портрет клиентов»** — упомянут, не приложен.

- **Кейсов с цифрами** — есть только один пример вскользь (Falcon, см. ниже). Основные кейсы на сайте + по «ссылкам для нового сайта» — не даны.
- **Регалий** (лет на рынке, число проектов, размер команды) — нет.
- **Контакта для подписи** — нет (Евгений — стратег/согласующий, не обязательно лицо подписи).
- **Оффера/гарантий** — Костя прямо сказал: «оффера нет нигде». Гарантии — под вопросом («может были какие-то интересные»).

## Что продаём и общая механика (единая для всех 3 цепочек)

- Продукт — **SEO-продвижение**, но **в письме SEO напрямую не продаём**.
- Задача цепочки — закрыть получателя на ответ **«интересно»** → дальше менеджер просит телефон, хвалит, закрывает на встречу, и **только потом** отдаёт ценность (аудит / план / стратегию).
- Важная вводная по механике ответа: получатели **ничего ответным письмом не высылают** — они просят номер телефона и ведут на звонок/встречу. Значит CTA письма = получить «интересно»/контакт, а не «вышлем вам прямо сейчас».
- Приманка (lead-magnet) во всех цепочках — **бесплатный аудит** (можно в каждом письме закрывать на разное: аудит / план действий по сайту / стратегия продвижения).

## 3 цепочки (структура от стратега)

### Цепочка 1 — Персонализированный аудит (сегмент: производства/заводы)

**Угол:** айсбрекер = персональный SEO-показатель по сайту получателя.

- **Механика айсбрекера (как делается на стороне агентства):** вычисляем производственные компании по сайту → разными инструментами снимаем показатели по SEO → добавляем **уникальный параграф** под конкретный сайт. Письмо мимикрирует под ручную проверку («проверили ваш сайт в Яндексе/Гугле руками»).
- **Каркас письма №1 (со слов стратега, дословная логика):**
  «[приветствие]. Проверили ваш сайт в Яндексе/Гугле *(конкретный показатель/инструмент дальше согласует Евгений)* и увидели вот такие уникальные параметры *(1 строка — персональный параметр)*. Похоже, по SEO сейчас теряете клиентов. Чтобы обогнать конкурентов — давайте проведём аудит и всё бесплатно расскажем. Интересно или нет?»
- **ЦА / ЛПР:** производства, заводы.
  - Завод **маленький** → руководитель/собственник.
  - Завод **большой** → директор по маркетингу / маркетолог.
  - Добавляем «передайте коллегам».
- **Цель цепочки (3 письма):** продать именно **аудит** (не SEO).
- **CTA:** «интересно?» → бесплатный аудит.

> ⚠️ Персональный SEO-параметр в письме = **плейсхолдер** (оператор подставляет реальные данные по сайту). Конкретные показатели не выдумывать. Формулировку «проверили в Яндексе/Гугле + какой именно параметр» **согласует Евгений** — открытое поле.

### Цепочка 2 — Триггер «вакансии» (любые отрасли)

**Угол:** видим, что компания нанимает сеошника/маркетолога → предлагаем рассмотреть аутсорс как альтернативу найму.

- **Триггер:** открытая вакансия (нужен сеошник / маркетолог / смежная роль). **Любые вакансии из любых отраслей.**
- **Аргументы за аутсорс (со слов стратега) — ключевое выделено:**
  - 🔑 **Насмотренность и экспертиза агентства** — через агентство проходит больше клиентов, чем видит один штатный спец; больше кейсов и опыта. *(стратег: «вот это ключевое на самом деле»)*
  - Легко начать / легко остановить (vs наём-увольнение).
  - По деньгам может быть доступнее.
  - Работает целая команда по цене одного специалиста.
  - **Безопасность: подписываем NDA** — важно для заводов.
- **CTA:** «интересно?» → переговорим, вышлем аудит / план действий по сайту / стратегию продвижения (в каждом письме можно закрывать на разное). По факту ценность отдаётся только на звонке/встрече.

> Бизнес-логика триггера C2 отличается от C1: C1 = «мы нашли проблему на вашем сайте», C2 = «вы и так решаете задачу наймом — есть способ выгоднее». Разводка C1/C2 — по триггеру и боли, не по лексике.

### Цепочка 3 — Матрица кейсов (под конкретный кейс/нишу)

**Угол:** «в вашей нише у нас уже есть результат» — пишем под нишу получателя, опираясь на похожий кейс.

- **Механика:** берём конкретный кейс → находим получателей в той же/похожей нише → пишем «к нам обратилась похожая компания, была вот такая проблема, решили, вот результаты — хотите так же / посмотреть как сделали / другие кейсы?».
- **Пример (единственный из инструкции):** кейс **Falcon — интернет-магазин свето-тюнинга**. Пишем либо под «свето-тюнинг», либо под «интернет-магазин», под похожие компании.
- **Правила подачи кейса (со слов стратега):**
  - **Упор на цифры/результат**, а не на описание процесса («как делали — всем пофиг»).
  - **Год не указывать.**
- **Оффер/гарантии:** «оффера нет нигде»; гарантии — под вопросом, уточнить.
- **CTA:** «напишите интересно — свяжемся» + упомянуть бесплатный аудит.

> ⚠️ Цифры кейсов — берём ТОЛЬКО из реальных кейсов клиента (сайт/документы), которых пока нет. Falcon как пример без подтверждённых цифр — в цепочку не идёт, пока не получим данные. C3 = матрица: по одному примеру цепочки на ключевой кейс/нишу.

## Формат

- Стандарт CLAUDE.md: **3 письма** в каждой цепочке, тема в каждом письме (одинаковая по сегменту, с `{Компания}`), приветствия/подписи — плейсхолдеры `[приветствие]` / `[подпись]`.
- Цепочка 3 — «матрица»: по примеру цепочки под ключевой кейс.

## Открытые поля (запросить ДО написания цепочек)

1. **Точное название бренда + домен сайта.** Без сайта нельзя ни проверить кейсы, ни сослаться на разделы, ни понять реальное УТП.
2. **Документ «портрет клиентов»** — для уточнения ЦА/сегментов и болей.
3. **Документ «УТП и выгоды»** — чем отстраиваются, кроме «насмотренности агентства».
4. **Кейсы с цифрами** (формат: Ниша / Проблема / Что сделали / Результат) — критично для Цепочки 3 и для усиления C1/C2. Ссылки «для нового сайта» + раздел «кейсы» на текущем сайте.
5. **Контакт для подписи** (имя/должность/телефон/почта менеджера).
6. **Есть ли гарантии/оффер** — стратег сомневается; уточнить, можно ли что-то обещать.
7. **Финальная формулировка айсбрекера C1** (какой именно SEO-показатель показываем и как) — согласует Евгений.
8. **Сегментация для C3** — какие ниши/кейсы приоритетны (пока только пример Falcon).

## Контекст (не для писем)

- Это голосовая инструкция Кости, не прямой созвон с клиентом. Продажный видеозвонок смотреть не нужно (указание Кости).
- Стратег Евгений согласует формулировки айсбрекера (особенно C1).
