Что продаем в аутриче (B2B-направление)

  1. Корпоратив / тимбилдинг — выездной корпоратив «под ключ», «всё включено». От 20 до 150-200 человек + выездные стратегические сессии для руководителей.
  2. Центр обучения — образовательные форумы, интенсивы, выездные офлайн-курсы онлайн-школ, soft-skills тренинги, бизнес-лагеря, авторские ретриты, турниры (квизы/шахматы/дебаты).
  3. Творческий ретрит — выездные сборы для музыкальных/вокальных школ, театральных и танцевальных коллективов, арт-школ, йога- и медитационных групп.
  4. Банкеты, юбилеи, праздники — корпоративные вечеринки, юбилеи, выпускные.
  5. Свадьбы «под ключ» (скорее B2C — кросс-сейл, не приоритет аутрича).
  6. Аренда залов — 5 площадок под конференции, форумы, концерты, выставки.

УТП (что вытаскивать в письмо)

  1. «Всё включено» = ноль забот для организатора — размещение, питание, залы, техоснащение, досуг и вечерние программы в одном пакете.
  2. Море + 1-я линия + собственный пляж — смена обстановки, которой нет у городских площадок.
  3. 2 часа от Краснодара — короткое плечо для группового выезда из ЮФО.
  4. Масштабируемость — от 20 до 500 человек, 5 залов под любой формат.
  5. Круглогодичность — всесезонный отдых, межсезонье дешевле.
  6. Единое пространство «живут вместе» — эффект комьюнити для обучения/ретритов.
  7. Доп.инфраструктура — SPA, баня у моря, аквапарк, дельфинарий, анимация без отдельной организации.
  8. Все на одной охраняемой территории (формулировка клиента) — залы, проживание и отдых в одном периметре, не нужно развозить участников по городу.

Сегмент 1 (от клиента, 2026-06-17) — «Прямые бизнес-заказчики»

Канонический сегмент от клиента — ядро аутрича. Расширение базы B2B-партнеров, которые отправляют работников отдыхать, проводят мероприятия и выезды. Шире «турбизнеса» из матрицы: прямые компании + профсоюзы + event-организаторы + туроператоры.
  • Кто (целевые отрасли): B2B-партнеры, отправляющие работников отдыхать; организаторы мероприятий (спортивных, танцевальных); профсоюзы (выезд для работников). Организации и предприятия — коммерческие и федеральные. Туроператоры и турагентства.
  • Целевые должности: ответственные за решения по выездам / отпускам / мероприятиям; MICE-менеджер.
  • Триггер: дать возможность провести выездные мероприятия на берегу моря по системе «всё включено», с конференц-залами и киноконцертным залом.
  • Боли: проблема с арендуемыми площадями в городах; желание попробовать новые места, где закрывается и потребность в залах, и есть отдых.
  • УТП / выгоды: удобная геолокация; 1-я береговая линия; вся инфраструктура на одной охраняемой территории.
  • География: ЮФО, Москва, МО.
  • Цикл сделки: 1-6 месяцев.
  • CTA / закрываем на: запрос КП / презентации; номер телефона; лид-магнит.
  • Проспектинг: организации и предприятия (коммерческие + федеральные), туроператоры, турагентства. Объем email — открытое поле.
  • Кейс-якоря (из матрицы): Дискавери / Интуравто / Глэдтур; МНТОРЭС, АИС; Ждановская ППО; Движение Первых. С разрешения называть.

CTA-лестница (под механику клиента: КП/презентация/телефон)

  1. Письмо №1 (низкий): короткий интерес — попросить прислать КП/презентацию по площадке.
  2. Письмо №2 (средний): запрос под формат — даты, число участников, какой зал → индивидуальный расчет / подбор площадки.
  3. Письмо №3 (тяжелый): телефон для разговора с менеджером / ознакомительный визит.
Закрываем на запрос КП/презентации или телефон (механика клиента), не на бронь напрямую. Уточнить: есть ли готовый лид-магнит/презентация для ответа.

Запрещено в письмах (CLAUDE.md)

  • Тема — в каждом письме (одинаковая по сегменту, с {Компания}); приветствия/подписи — только плейсхолдеры [приветствие] / [подпись]
  • Антитезы «не X, а Y»
  • Буква «е» (всегда «е»); длинное тире — (использовать –); дефис - для диапазонов цифр
  • Любая выдуманная конкретика (цифры эффекта, B2B-кейсы, причины боли, цены, варианты ответа) — только из брифа/сайта
  • Каждая цифра в письме = ссылка на источник в блоке «Источники» под цепочкой

Следующий шаг

С1 определен клиентом — можно стартовать цепочку. До запуска нужны 3 вещи:

  1. Контакт/подписант под B2B (куда уходит лид, чьим именем подпись).
  2. Лид-магнит/презентация/КП — что шлем в ответ на интерес (CTA на него закрывает).
  3. Разрешение называть кейсы из матрицы.

Затем: Pre-flight + первая цепочка по С1 через пайплайн копирайтер → ревьюер. CTA = запрос КП/презентации/телефона. Тему держать на проблем-языке («арендуемые залы в городе»); Туапсе/дроны в письмо не выносить.
