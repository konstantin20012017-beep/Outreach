## Что подтвердить у клиентов перед публикацией

1. Отрасль и точное название (или согласие на анонимный кейс).
2. Бизнес-задача (раздел «Задача `[подтвердить]`» в каждом файле).
3. Официальную цитату-тестимониал.
4. Разрешение публиковать абсолютные цифры (или заменить на относительные: «+X%», «в N раз»).
