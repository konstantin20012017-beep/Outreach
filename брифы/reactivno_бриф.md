## Три цепочки (от маркетолога) — разводка по триггеру

### Сегмент 1 — Персонализированный аудит (триггер: данные по SEO)
- **Угол:** айсбрекер = персональный SEO-показатель по сайту получателя («проверили ваш сайт руками»).
- **Механика айсбрекера:** база с сайтами → ИИ Coldy + **Топвизор (% видимости в Яндексе)** → уникальный параграф под сайт. Образец опенера: *«Проверили ваш сайт в Яндексе, видимость средняя 34%. Похоже, по SEO теряете клиентов; чтобы составить конкуренцию — предлагаем индивидуальный аудит.»*
- **ЦА:** Производство, Заводы. Маленький завод → руководитель/собственник; большой → директор по маркетингу/маркетолог. Добавлять «передайте коллегам».
- **Цель 3 писем:** продать именно **аудит**.
- ⚠️ Персональный SEO-параметр в письме = **плейсхолдер** (оператор подставляет реальные данные: Топвизор/Coldy). Конкретику не выдумывать. Точную формулировку «какой показатель» **согласует Евгений**. **Открытый вопрос — ии-аудиты (Вадим).**

### Сегмент 2 — Триггер «вакансии» (триггер: открытая вакансия)
- **Угол:** видим вакансию (SEO / маркетолог / руководитель маркетинга) → продаём идею аутсорса как альтернативы найму, кратко, с указанием с кем переговорить.
- **Аргументы за аутсорс (ключевое выделено):**
  - 🔑 **Насмотренность и экспертиза агентства** — через агентство проходит больше клиентов/кейсов, чем видит один штатный спец *(стратег: «вот это ключевое»)*.
  - Легко начать / легко остановить (vs наём-увольнение).
  - По деньгам может быть доступнее; работает команда по цене одного.
  - **NDA** — важно для заводов.
- **ЦА:** любые отрасли с вакансией. Целевые из таблицы: услуги для бизнеса (бухгалтерия, юр.услуги, HR), частные клиники и медцентры. Должности: руководитель / директор по маркетингу / маркетолог.
- **CTA:** «интересно» → переговорим → аудит / план по сайту / стратегия (на звонке).

### Сегмент 3 — Матрица кейсов (триггер: кейс под ОКВЭД получателя)
- **Угол:** «в вашей нише у нас уже есть результат» — пишем под нишу получателя, опираясь на похожий кейс с цифрами.
- **Механика:** берём кейс → находим получателей в той же/похожей нише → «к нам обратилась похожая компания, была такая проблема, решили, вот цифры — хотите так же / посмотреть как сделали / другие кейсы?».
- **Правила подачи:** упор на **цифры/результат**, не на процесс. **Год не указывать.** Имя клиента-кейса — по согласованию (на сайте кейсы анонимны по нише).
- **Маркетолог особо просит:** под каждый сегмент/получателя подбирать кейс под конкретную нишу.
- **CTA:** «напишите интересно — свяжемся» + бесплатный аудит.
