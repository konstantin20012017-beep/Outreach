Бриф – ExportBase

export-base.ru. Создан 2026-05-12.

Сайт: export-base.ru
Продукт: базы контактов B2B-компаний с гибкими фильтрами (разовые выгрузки, без подписки).
На рынке: с 2014 (опыт работы с маркетинговыми агентствами).

Ключевая фича

Бренд + ИНН + ФИО директора одной записью. У конкурентов (Контур.Компас) – только юрлица.

УТП

  • Стоимость от 1 копейки за контакт (чем больше выборка – тем дешевле).
  • Разовые выгрузки без подписки.
  • Фильтры: ОКВЭД, регион, размер, маркетплейсы (WB/Ozon), лицензии, товарные знаки.
  • Погрешность по email ~5%.

ЦА

Маркетолог (или РОП), руководитель подразделения, собственник. Россия, ЦФО – приоритет.

CTA

Связь с персональным менеджером / ознакомление с сайтом.

Сегменты под цепочки (10 гипотез консультанта)

1-я очередь:

  • Сегмент 5 – Реанимация своей базы (~4500). Сегментация: (а) зарегались без оплаты, (б) выгружали год+ назад, (в) активные – не трогать.
  • Сегмент 1 – Маркетинговые агентства. Кейс-якорь: Комплето.
  • Сегмент 4 – Агентства маркетплейсов. Уникальная фича: база продавцов WB/Ozon.

2-я очередь: Performance-агентства (Я.Директ), колл-центры, email-аутрич агентства (кейс Евклид).
3-я очередь: IT/SaaS, HR-агентства, юристы, производственники.

Готовые цепочки

  • Реанимация (сегмент 5): v1 – цепочки/exportbase_реанимация_v1.html
  • Маркетплейсы (сегмент 4): v1 – цепочки/exportbase_маркетплейсы_v1.html

Открытые поля (запросить у клиента)

  • ФИО / телефон / email менеджера для подписи.
  • Финальный список приоритетных отраслей по сегментам.
  • Выгрузка сегментов своей базы (зарегались без оплаты / неактивные год+).
  • Домены для регистрации, паспортные данные.
  • Реальные кейсы с цифрами (по агентствам, колл-центрам, МП).
  • Подтверждение объёма бесплатного теста (50 контактов взято из паттерна консультанта).
  • Подтверждение скидки 30% на первую выгрузку после паузы (трипваер из стратегии).
  • Скриншот формата записи МП-базы – пригодится для №2 как артефакт.
  • Уточнить про категории МП в кабинете – есть ли фильтрация по нише товара WB/Ozon помимо ОКВЭД.
