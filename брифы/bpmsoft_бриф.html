Структура работы — 3 цепочки на одну нишу

Все 3 цепочки идут на одну и ту же ЦА. Цель — A/B-тест: проверить, какой заход даст больше регистраций.

Цепочка №1 — заход «по референсу LeadBridge» — v1 готова

  • Файл: цепочки/bpmsoft_цепочка1_рефLeadBridge_v1.html
  • Формат: 2 письма с темой в теле, плейсхолдеры {{firstName}} + [приветствие]. Ссылка на регистрацию НЕ в письме — отправляется отдельно тем, кто ответил «+»
  • Письмо №1: «Приглашение на вебинар BPMSoft, 26 мая» + общая программа + вопрос «есть ли интерес»
  • Письмо №2: «Поднимаю переписку» + конкретика по спикерам (ВОЛМА x5, SEB +30%, Петров от BPMSoft) + повтор вопроса
  • Источник: Coldy/LeadBridge — 49 регистраций за 2 недели, 163% от плана

Цепочка №2 — заход через бренды и регалии — v2 готова

  • Файл: цепочки/bpmsoft_цепочка2_бренды_v2.html (v1 устарела)
  • Формат: 3 письма, ссылка в каждом
  • Письмо №1: регалии (1 место TAdviser/CNews/Сколково) + список 14 брендов (ГАЗ, ВОЛМА, ЦУМ, VK, Билайн, Газпромнефть, ТеДо, Б1 и др.). Боль: зоопарк систем
  • Письмо №2: спикеры (ВОЛМА — лиды -40%, закупки -20%, КП с дней до 2 часов; SEB +30%) + блок других клиентов: Билайн +24% / 4000 чел.-часов, VK -60% отчёты, Аскона x1.5 онлайн, Туту 3 часа перенастройки. Боль: скорость БП и закупок
  • Письмо №3: короткий follow-up + PS «перешлите коллеге»

Цепочка №3 — заход через миграцию + ROI — v2 готова

  • Файл: цепочки/bpmsoft_цепочка3_миграция_ROI_v2.html (v1 устарела)
  • Формат: 3 письма, ссылка в каждом
  • Письмо №1: регалии → реальная боль (Salesforce/Dynamics/SAP ушли + КИИ + 152-ФЗ) → миграции с конкретикой (ЦУМ за 2 недели без потери данных, ТеДо 750+ мест, Б1 1052 места, ППР/Биокодекс/GEFCO/СберФакторинг с сохранением исторических данных) + ФСТЭК. Боль: импортозамещение
  • Письмо №2: контекст про BPMSoft → 5 ROI-цифр с привязкой к кейсам (цикл x2-3 → ППР, конверсия x1.5 → Аскона, чек +24% → Билайн, отчёты -60% → VK, low-code 3 часа → Туту) + усреднённые (маркетинг -50%, IT -40%) + спикеры. Боль: окупаемость и стоимость владения
  • Письмо №3: короткий follow-up + PS «перешлите коллеге, отвечающему за CRM или импортозамещение»

Формат писем (по правилам проекта)

  • Без темы (исключение — цепочка №1 по референсу LeadBridge: тема — часть формата кейса)
  • Приветствие/подпись — [приветствие] / [подпись]
  • Без буквы «ё» (всегда «е»)
  • Тире «–», диапазоны цифр через дефис «-»
  • Без антитезы «не X, а Y»
  • HTML, Arial 11pt, заголовки h2 18pt bold, line-height 1.15
