# ExportBase – исходники

Источники: бриф-таблица клиента + стратегический документ внешнего консультанта (10 гипотез GTM-цепочек, апрель 2026).

## Бриф клиента (поля)

**Компания:** ExportBase, export-base.ru
**Продукт:** Базы контактов потенциальных клиентов (компаний/юр.лиц)

**Целевые должности:** Маркетолог (если нет – РОП), руководитель подразделения, собственник.

**Почему пишем (триггеры):**
- потребность в расширении клиентской базы
- актуализация уже существующей базы клиентов
- подбор ЦА для таргетинга в интернет-рекламе

**Боли:**
- мониторинг новых потенциальных клиентов
- устаревание информации в текущих БД клиентов
- низкая точность нацеливания в рекламных кампаниях (Я.Директ)

**УТП и выгоды:**
- низкая удельная стоимость контакта – 1 копейка и даже меньше копейки (цена зависит от количества фильтров; чем больше компаний в выборке – тем дешевле)
- высокая гибкость фильтров для точного подбора аудитории
- опыт сотрудничества с маркетинговыми агентствами с 2014 года

**Целевое действие:** ознакомление с сайтом / связь с персональным менеджером.

**География:** Россия, ЦФО – приоритет.

**Проспектинг:** сайты компаний, гос. источники.

**Своя база для реанимации:** есть.
**CRM:** нет.
**Каналы трафика:** Я.Директ, ВК, прямые и поисковые заходы.

## Карточка из анализа консультанта

- Уникальная фича (из созвона): связка бренд + ИНН + ФИО директора в одной записи.
- Дополнительно: фильтр по маркетплейсам (WB/Ozon), наличие лицензий, зарегистрированные товарные знаки, телефоны и email.
- Своя база для реанимации: ~4500 компаний.
- Конкурент-ориентир: Контур.Компас (ExportBase выгоднее – нет подписочной модели, разовые выгрузки).
- Погрешность по email базах: ~5% (формулировка из гипотезы 3).

## Сегментация базы реанимации (из стратегии)

1) Зарегались, но не пополнили баланс.
2) Когда-то выгружали, год+ не активны.
3) Активные – НЕ трогать.

## Кейсы-имена, упомянутые в материалах

- **Комплето** – маркетинговое агентство, постоянный клиент, регулярно берёт базы под клиентские кампании.
- **Евклид** – упомянуто как «email-аутрич агентство» (контекст – гипотеза 6).
- ExportBase сам использует Я.Директ + базы для ретаргетинга в собственном маркетинге.

## Открытые поля (запросить у клиента)

- Целевые отрасли по сегментам (клиент пришлёт).
- Оценка количества потенциальных email.
- ФИО, телефон, email менеджера для подписи (+ сайт, TG?).
- Список доменов для регистрации.
- Паспортные данные / организация для регистрации доменов.
- Сегментация собственной базы (выгрузка из своей панели).
