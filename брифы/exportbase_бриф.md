# Бриф – ExportBase

**Сайт:** export-base.ru
**Продукт:** базы контактов B2B-компаний с гибкими фильтрами. Есть разовые выгрузки и подписка – клиент выбирает формат.
**На рынке:** с 2013 года (бриф клиента отдельно упоминает «с 2014 с маркетинговыми агентствами» – это специализация, не дата основания).

## Регалии (с сайта)

- 50 000+ клиентов, 140 000+ покупок.
- 464 готовых тематических базы в Магазине.
- 37 открытых гос. источников: ФНС, ФАС, Росстат, Федресурс, ФССП, ФИПС, ЕИС Закупки, МЧС и др. Снимает вопрос легальности.
- База обновляется ежедневно.
- 6 месяцев бесплатных обновлений базы после покупки.
- Только юрлица из открытых источников, без персональных данных физлиц (не нарушает 152-ФЗ).
- Юрлицо: ИП Колтыгина С. А., Ярославль.

## Ключевая фича

Бренд + ИНН + ФИО директора одной записью. У конкурентов (Контур.Компас) – только юрлица. На сайте формулировка «Связка Бренд + Данные ЕГРЮЛ».

## УТП

- Стоимость от 1 копейки за контакт (чем больше выборка – тем дешевле).
- Гибкость фильтров: ОКВЭД, регион, размер, правовая форма, маркетплейсы (WB/Ozon), лицензии, товарные знаки, выручка, численность сотрудников и др.
- Look-alike – выборка по параметрам существующих клиентов.
- Готовые тематические базы в Магазине: продавцы WB+Ozon+ЯМ, HoReCa, стоматологии, аккредитованные ИТ-компании, агропром и др.
- Форматы выгрузки: XLSX, CSV (включая 1С-Битрикс), SQL, JSON.

## Дополнительные услуги

- Колл-центр для холодного обзвона.
- Парсинг сайтов и мониторинг цен на заказ.
- API доступ.
- Обогащение существующей базы по ИНН/ОГРН.

## ЦА

Маркетолог (или РОП), руководитель подразделения, собственник. Россия, ЦФО – приоритет.

## Сегменты под цепочки (по стратегии консультанта, 10 гипотез)

**1-я очередь:**
- **Сегмент 5 – Реанимация своей базы (~4500).** Сегментация: (а) зарегались без оплаты, (б) выгружали год+ назад, (в) активные – не трогать.
- **Сегмент 1 – Маркетинговые агентства.** Кейс-якорь: Комплето (постоянный клиент).
- **Сегмент 4 – Агентства маркетплейсов.** Уникальная фича: готовая база продавцов WB/Ozon/ЯМ как отдельный продукт + фильтр в конструкторе.

**2-я очередь:** Performance-агентства (Я.Директ – заход через look-alike), колл-центры, email-аутрич агентства (кейс Евклид).
**3-я очередь:** IT/SaaS, HR-агентства, юристы (заход через лицензии/тендеры/знаки), производственники (заход через дилеров).

## Цепочки – статус

- **Реанимация (сегмент 5):** v1 готова – `цепочки/exportbase_реанимация_v1.html`. 3 цепочки (А новизна+скидка, Б разбор прошлого опыта, В альтернатива ручному сбору).
- **Маркетплейсы (сегмент 4):** v1 готова – `цепочки/exportbase_маркетплейсы_v1.html`. 3 цепочки (А уникальная база, Б вопрос-крючок, В как дотянуться до новых селлеров).

## Открытые поля (запросить у клиента)

- ФИО / телефон / email менеджера для подписи.
- Финальный список приоритетных отраслей по сегментам.
- Выгрузка сегментов своей базы (зарегались без оплаты / неактивные год+).
- Домены для регистрации, паспортные данные.
- Реальные кейсы с цифрами (по агентствам, колл-центрам, МП) – на сайте только общие 50 000 клиентов / 140 000 покупок, конкретных кейсов «компания X получила результат Y» нет.
- Подтверждение объёма бесплатного теста (50 контактов взято из паттерна консультанта).
- Подтверждение скидки 30% на первую выгрузку после паузы (трипваер из стратегии).
- Скриншот формата записи МП-базы – пригодится как артефакт в письме №2.
- Уточнить про категории МП в кабинете – есть ли фильтрация по нише товара WB/Ozon помимо ОКВЭД.
