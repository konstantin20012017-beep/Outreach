# Промпоставки — каркасы цепочки под госсектор/корпорации (официальный язык)

> Референс-каркасы под требование маркетолога (2026-06-08): официальный деловой тон, адресат – госкомпании/корпорации, одна цепочка, один регион.
> Это СКЕЛЕТ с плейсхолдерами – не финал. Конкретику (продукт, кейсы, регалии, регион) подставляем после брифа по сегменту. Цифры/кейсы не выдуманы – везде плейсхолдеры.

## Что меняется в B2G vs наш обычный аутрич

| Ось | Обычный аутрич | Госсектор/корпорации |
|–|–|–|
| Тон | живой менеджер, «вы», лёгкость | официальное деловое письмо, нейтрально-уважительно |
| Обращение | [приветствие] | «Уважаемый(ая) [имя отчество]» / «Уважаемые коллеги» |
| Самопредставление | мягко, по ходу | сразу: кто компания, чем занимается, без воды |
| Подпись | имя + контакт | ФИО, должность, компания, ИНН, телефон, email, сайт |
| CTA | «ответьте да/интересно» | направить КП / включить в перечень поставщиков / готовность участвовать в закупках |
| Цель письма | разговор | попасть в пул поставщиков под будущие закупки 44-ФЗ/223-ФЗ |

**Ключевое про механику:** прямую продажу госзаказчику холодным письмом не закрыть – закупки идут через тендеры. Реалистичный next step: предложить себя как поставщика на будущие закупки, направить коммерческое предложение / прайс для ознакомления, попасть в реестр потенциальных поставщиков подразделения снабжения. CTA строим под это.

## Структура официального письма (по абзацам)
1. Обращение + кто мы (компания, профиль, регион присутствия).
2. Зачем пишем (предлагаем как поставщика по [категория] для нужд [тип организации]).
3. Что предлагаем + почему мы (регалии/опыт/номенклатура – из брифа).
4. Конкретный next step (направить КП / включить в перечень / готовность к закупкам).
5. Подпись с реквизитами.

⚠ Запреты проекта сохраняются: без «ё», тире `–` для текста, дефис `-` для диапазонов, без антитез «не X, а Y», тема одинаковая во всех 3 письмах с `{Компания}`.

## КАРКАС цепочки (3 письма, официальный тон, плейсхолдеры)

**Тема (одна на все 3):** `[тема: предложение поставщика по {категория} – {Компания}]`

### Письмо №1 — представление и предложение
Тема: [тема выше]

[приветствие официальное]

Компания [название клиента] – [профиль: чем занимается] с поставками в [регион]. Мы поставляем [категория продукции/услуг] для нужд организаций [тип: госучреждения / корпорации / промпредприятия].

Направляем предложение для рассмотрения [Компания] в качестве поставщика по направлению [категория]. [1 предложение: ключевая регалия/опыт из брифа – плейсхолдер].

Готовы направить коммерческое предложение и номенклатуру с актуальными условиями. Подскажите, на какой адрес и в адрес какого подразделения корректно направить материалы для включения в перечень потенциальных поставщиков.

[подпись с реквизитами]

### Письмо №2 — опыт/номенклатура + конкретика
Тема: [та же]

[приветствие официальное]

Возвращаемся к вопросу о сотрудничестве по направлению [категория].

[Опыт под тип заказчика – плейсхолдер из брифа: с какими организациями/отраслями работали, что поставляли. БЕЗ выдуманных цифр.] Работаем по [условия: договор/тендеры 44-ФЗ/223-ФЗ – уточнить у клиента], полный пакет документов.

Если в [Компания] планируются закупки по данному направлению, готовы предоставить коммерческое предложение с учётом ваших требований и техзадания. Сообщите, пожалуйста, контактное лицо отдела снабжения для направления материалов.

[подпись с реквизитами]

### Письмо №3 — финальное, фиксация на перспективу
Тема: [та же]

[приветствие официальное]

Завершаем серию писем по вопросу поставок [категория].

Понимаем, что закупки планируются заранее и под конкретные процедуры. Просим рассмотреть возможность включения [название клиента] в перечень поставщиков [Компания] по направлению [категория] – чтобы при формировании потребности мы могли оперативно направить предложение.

Для этого достаточно подтвердить контакт ответственного подразделения. Будем признательны за ответ.

[подпись с реквизитами]

## CTA-лестница (официальная, вместо «ответьте да»)
- №1 (лёгкий): «подскажите адрес/подразделение для направления материалов».
- №2 (средний): «сообщите контактное лицо отдела снабжения для КП под ваше ТЗ».
- №3 (закрывающий): «рассмотрите включение в перечень поставщиков – подтвердите контакт».

## Чего избегать в B2G (убивает письмо)
- Фамильярности, эмодзи, восклицаний, «горящих» скидок.
- Обещаний/цифр без основания (госадресат проверяет).
- Давления и манипуляций срочностью.
- Спам-триггеров: «бесплатно!!!», «только сегодня», КАПС, много ссылок.
- Антитез и рекламного самовосхваления.

## Открытые поля для финала (из брифа)
- Регион (один крупный) – ждём данные клиента.
- Сегмент в госсектор: сетка / оборудование / логистика.
- Регалии/опыт/номенклатура клиента, кейсы с госзаказчиками.
- Реальная механика сделки (тендер / прямой договор / КП в пул) → уточняет CTA.
- Реквизиты для подписи + от чьего лица.
