Механика исполнения рассылки (созвон с подрядчиком, 2026-06-05)

Источник: транскрибация созвона подрядчика email-аутрича (Вадим, агентство Eflipt) — в аптека_исходники.md. Привязана к АСНА как разбор метода. Цены/ЦА/кейсы из созвона — контекст подрядчика, НЕ факты АСНА.
  • Формат: 3 письма на контакт, у каждого свой триггер и свой CTA (= наша CTA-лестница).
  • CTA = регистрация через форму с телефоном. Подрядчик считает лидом «интерес + номер телефона»; форма АСНА просит телефон → CTA «зарегистрируйтесь» сходится с моделью лида.
  • Канал — email-аутрич (не ТГ): прогрев ~2 недели, результат ~3 недели рассылки. Воронка-ориентир: open rate №1 ~25% / №2 ~10% / №3 ~5%, ответы ~2%.
  • Гипотеза усиленной персонализации (открытый вопрос): ответившим на №1 во 2-м касании отдают индивидуальный разбор (в созвоне — AI-разбор сайта/ниши). Применять ли к АСНА — решить с пользователем; для аптечной сети «разбор сайта» спорен, чем заменить — не домысливать.
